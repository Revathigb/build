Data editor

This editor let’s you adjust the data shown in the grid.
  • The initial contents is the actual data shown in the grid instance above it.
  • After editing the data, bind it to the grid by clicking grid.setData(…).
  • The grid in turn is bound to the data window: Any edits made to cells (via cell editors) will alter the contents of this editor.

In actual practice, when data is set using setData, it typically has come from a data store, typically as a character string in JSON format. If that is the case, you would probably call JSON.parse on it, rather than eval; and pass the result to setData. In this workbench, we’re using straight JavaScript, not JSON, which is why we’re using using eval, not JSON.parse. When editing, therefore, you do not have to worry about strict adherence to the JSON standard (double quotes, primitive types, _etc._).
