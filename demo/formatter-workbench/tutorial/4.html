Editor pane

The four tabs in the lower left pane are for editing data and code.
  • Apply your edits by clicking the grey button in the upper right of each tab.
  • The button’s label indicates the API call involved.
  • The API is called with the eval’d editor text.

Data tab

The Data editor is provided to adjust the data to suit the needs of your localizers and cell editors.

State tab

This tab lets you set grid properties declaratively.

Localizer tab

Value formatting in Hypergrid is controlled by localizers.

Cell Editor tab

This tab lets you create and edit cell editors. It comes with a sample cell editor called Time for editing values formatted with the hh:mm localizer.
