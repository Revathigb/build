Adding a new localizer

For a grid cell to use a localizer to format its value, three coding steps are required:
  1. Define the localizer
    Code the localizer in the Localizers tab's editor.
  2. Register the localizer with the grid
    Use the Localizers tab’s grid.localization.add(…) button.
  3. Bind cell(s) to the localizer
    Cells are loosely bound by name to the localizer using the format property. Use the State tab's editor to set the format property declaratively, either for the whole grid or for a specific column or cell.
